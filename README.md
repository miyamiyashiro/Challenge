Loja ItemsSuperMegaImportantes

Este é sistema interativo em Swift inspirado em Baldur's Gate III, criado para o desafio final da etapa cinco do processo seletivo do programa da Apple Academy SENAC.

O projeto consiste em uma loja medieval onde o jogador(Tav) pode visualizar itens, comprar equipamentos, gerenciar seu ouro e acessar seu inventário. Dependendo do seu nivel, ele pode tambem, conseguir um desconto. Tudo é feito diretamente pelo terminal, utilizando apenas os conceitos básicos de Swift ensinados durante workshop.

Este projeto utiliza os principais conteúdos vistos no workshop:

- Variáveis e constantes  
- Arrays  
- Tuplas  
- If, else
- Switch 
- While
- Funções  
- Entrada de usuário com readLine()
- Impressão com print()
